# axum_socket


## Game Server
- 주로 tcp나 udp프로토콜을 이용해 네트워크 라이브러리로 데이터를 주고 받음

## 패킷
- 패킷은 서버와 클라이언트간 약속된 메세지 구성
- [해더 + 패킷타입 + 데이터] 로 구성되 있음
- 헤더:byte 배열의 크기


## RabbitMq
