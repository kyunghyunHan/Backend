# SIMD

💾 CPU 안에 있는 SIMD 명령어들:

1. SSE (128비트)
   - 한 번에 4개의 32비트 숫자 처리
   - [num1, num2, num3, num4] ← 128비트

2. AVX (256비트)  
   - 한 번에 8개의 32비트 숫자 처리
   - [num1, num2, num3, num4, num5, num6, num7, num8] ← 256비트

3. AVX-512 (512비트)
   - 한 번에 16개의 32비트 숫자 처리
   - 더 많은 숫자를 동시에!
